# Трекинг парковочных мест

## Окружение 
     Python 3.8.10

## Как начать

* `git clone git@github.com:vyunolbek/parking_tracking.git parking_track`
* `cd parking_track`
* `python -m venv venv`
* `venv\Scripts\activate`
* `pip install -r requirements.txt`
